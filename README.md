# Workly AI by CAPACITI

Workly AI is a modern, responsive SaaS-style web application designed to serve as an all-in-one **AI Workplace Productivity Assistant**. Built with a premium dark-themed interface, the platform empowers professionals to automate repetitive tasks, optimize their workflows, and reclaim hours of productivity every week.

---

## 🚀 Key Features

Workly AI includes a central dashboard overview and currently implements the following core productivity modules:

### 1. 📧 Smart Email Generator
* **Draft in Seconds:** Generate professional, polished emails instantly based on minimal prompt inputs.
* **Tone Control:** Built-in customization to match various workplace contexts, including formal, friendly, and persuasive tones.

### 2. 📝 Meeting Notes Summarizer
* **Transcript Transformation:** Turn long, raw meeting transcripts into structured summaries.
* **Actionable Insights:** Automatically extracts critical action items, decisions, and deadlines so no detail is lost.

### 3. 📅 AI Task Planner
* **Smart Prioritization:** Automatically organizes daily or weekly schedules using the Urgency vs. Importance matrix.
* **Optimized Planning:** Helps users block out time efficiently to tackle high-impact goals.
